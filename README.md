# Türkçe Projesi

This project is a simple web application consisting of an HTML file, a JavaScript file, and a CSS file for the Turkish Project. It serves as a basic example demonstrating fundamental web development structure, functionality, and styling concepts.

## Files

- `index.html`: The main HTML file rendering the web page.
- `script.js`: The JavaScript file containing interactive behavior.
- `style.css`: The CSS file for styling the web components.

## Usage

To view the project, open `index.html` in any modern web browser. No additional setup is required.
